# X Tournament Director — Evidence Coach 3.5

## Dependent loose-part selector

1. Choose the part category.
2. The part dropdown shows only parts in that category.
3. Filter large categories without closing the mobile keyboard.
4. Select the part.
5. Catalog role, Bit code, Ratchet height, and integrated-Ratchet metadata are retained automatically.

The list is generated from the app's Known Sets catalog. `Other / not listed`
remains available for a custom part.
