# X Deck Lab 3.0.0

A root-only, mobile-first Beyblade X tournament coach for GitHub Pages.

## Main workflow

1. Add owned products or loose parts.
2. Build or generate a legal three-Bey deck.
3. Follow the single highest-information owned-opponent mission.
4. Record the battle and ordinary Self-KO Yes/No answer.
5. Use Coach to see readiness, patterns, strengths, weaknesses, and the next mission.

## v3 coach system

- Player mode is the default. Technical analytics remain collapsed.
- Advanced mode opens evidence gates, matchup matrices, engineering proxies, order optimization, and forecasts.
- Home shows the current roadmap step.
- Coach presents one mission rather than an unexplained analysis dashboard.
- Test planning ranks legal, simultaneously owned opponents by information value, uncertainty, matchup deficit, meta relevance, and controlled self-KO needs.
- Attack-bit versus attack-bit tests remain excluded by default.
- Battle history produces plain-language coaching patterns.
- Deck generation optimizes the complete three-Bey lineup under legality and inventory constraints and explains why each shortlist is useful.

## Deployment

Upload every file in the release ZIP directly to the GitHub repository root. Enable GitHub Pages for the root branch. No build step or dependency installation is required for deployment.
