# Installation

1. Delete the current app files from the GitHub repository.
2. Keep `CNAME` only if a custom domain is used.
3. Extract the ZIP.
4. Upload all extracted files directly to the repository root.
5. Keep GitHub Pages configured for `main` and `/ (root)`.
6. Reload the project-site URL.

The storage namespace is unchanged. Existing inventory and evidence should
remain available.
