# Installation

Yes—delete the old app files and folders before uploading this edition.

## Keep only when applicable

- `.git` in a local cloned repository
- `CNAME` when using a custom domain

## Upload to the repository root

All files from this package go directly beside `index.html`. Do not create an
extra folder and do not upload the ZIP itself.

## GitHub Pages setting

- Source: Deploy from a branch
- Branch: main
- Folder: / (root)

## Cache cleanup

The app removes older service workers automatically. When an old interface
still appears, clear the site data for the GitHub Pages address and reload.
