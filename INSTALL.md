# Installation

1. Delete the current app files from the GitHub repository.
2. Keep `CNAME` only when using a custom domain.
3. Extract this ZIP.
4. Upload all extracted files directly to the repository root.
5. Keep GitHub Pages set to `main` and `/ (root)`.
6. Confirm the lower-right marker reads **Mobile build 2.4**.

Clear the GitHub Pages site's Safari website data if an older build remains.
